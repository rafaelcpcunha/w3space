Observação
Apesar do chat estar em uma página externa, iniciando ele com um usuário válido, todos diálogos onde há pesquisa, essa pesquisa é feita a partir do número identificador do usuário. Então funcionará de forma correta.

Para a conversa funcionar é necessário usar algum usuário válido na base de HLG. Ao contrário alertará a mensagem Infelizmente, ocorreu um erro do sistema... Esse cuidado só é por conta do teste estar sendo feito aqui. Pois o chat sendo iniciado a partir do portal, o usuário estará logado. O que impede esse problema.
